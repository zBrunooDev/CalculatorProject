# 📌 Calculadora Financeira — C# (Console)

Projeto criado para praticar lógica de programação, classes, métodos, cálculos matemáticos e separação de responsabilidades.

## 🚀 Funcionalidades
- Cálculo de juros simples
- Cálculo de juros compostos
- Simulação de economia mensal
- Menu interativo no console
- Entrada e validação de dados

## 🧠 Tecnologias
- C#
- .NET 8
- Programação Orientada a Objetos (POO)

## ▶️ Como executar
1. Clone o repositório:
   git clone github.com/zBrunooDev/CalculadoraFinanceira.git

2. Acesse a pasta:
   cd CalculadoraFinanceira

3. Execute:
   dotnet run
